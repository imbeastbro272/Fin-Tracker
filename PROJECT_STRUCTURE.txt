Typical Workflow
================

Step 1: Training (Python)
--------------------------
    Run your_training_script.py
         ↓
    Add export code at end (from complete_workflow_example.py)
         ↓
    Generates: tree_model.json, preprocessing.json


Step 2: Prepare Upload (Python)
--------------------------------
    cd ESP32_SmartLight
    python upload_to_spiffs.py
         ↓
    Validates files, creates data/ folder
         ↓
    Ready for SPIFFS upload


Step 3: Upload to ESP32 (Arduino IDE)
--------------------------------------
    Open ESP32_SmartLight.ino
         ↓
    Tools → ESP32 Sketch Data Upload  (uploads data/)
         ↓
    Sketch → Upload                   (uploads firmware)
         ↓
    Serial Monitor → Verify model loaded


Step 4: Test & Monitor (Real-time)
-----------------------------------
    LED brightness adjusts automatically
         ↓
    Turn potentiometer to adjust manually
         ↓
    System learns from adjustments >5%
         ↓
    Model retrains after 50 samples

Quick Reference Commands
========================

Python:
-------
# Export model
python your_training_script.py

# Prepare SPIFFS
cd ESP32_SmartLight
python upload_to_spiffs.py

Arduino IDE:
-----------
# Upload SPIFFS data
Tools → ESP32 Sketch Data Upload

# Upload firmware
Sketch → Upload (or Ctrl+U)

# Monitor output
Tools → Serial Monitor (115200 baud)

PlatformIO (Alternative):
-------------------------
# Build
pio run

# Upload firmware
pio run --target upload

# Upload SPIFFS
pio run --target uploadfs

# Monitor
pio device monitor
